## Featured Project

### Local Diary LLM
Privacy-focused desktop diary application powered by a locally running LLM.

- Built with Python and PyQt6
- Local LLM execution via Ollama (CLI integration)
- No external API dependency
- File-based persistence (no database)
- Chat mode, summarization, prompt editing
- PDF export and theme switching
